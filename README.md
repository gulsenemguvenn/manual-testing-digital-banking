\# Manuel Test – Dijital Bankacılık Projesi



Bu repo, bir \*\*dijital bankacılık uygulaması\*\* için uçtan uca \*\*manuel test yaklaşımını\*\* göstermektedir.

Proje kapsamında gereksinim analizi, risk tabanlı test yaklaşımı, test planı, test senaryosu yazımı,

hata raporlama ve test özet raporu çalışmaları yer almaktadır.



\## Proje Kapsamı

\- Giriş (Login)

\- Hesap Görüntüleme

\- Para Transferi

\- İşlem Geçmişi



\## Klasör Yapısı

\- `requirements/` → Fonksiyonel gereksinimler

\- `risk-analysis/` → Risk analizi

\- `test-plan/` → Test planı

\- `test-cases/` → Manuel test senaryoları

\- `bug-reports/` → Hata raporları

\- `test-summary-report/` → Test sonuç özeti
